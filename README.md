# Лабораторные работы по ООП (C#)

В проекте реализованы лабораторные работы 1–7 по объектно-ориентированному программированию на языке C#.

## Запуск

Откройте файл `Program.cs` и отредактируйте метод `Main()`:

```csharp
static void Main() {
    System.Console.WriteLine("[INFO] Start");

    // Оставьте активной только нужную лабораторную работу:
    Lab7.Lab7.Run_1();  // ← текущая работа

    System.Console.WriteLine("[INFO] End");
}
